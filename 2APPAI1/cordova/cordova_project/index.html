General information

Girls' Frontline Information Bank is a wiki like app, you can look up the general information of a T-doll and view there stats (and soon hopefully even more). You can easily favorite your favorite T-dolls for easy lookup and search T-dolls by build time so you know wish doll is in production.

What you can expect
  • View general information (T-dolls)
  • View stats (T-dolls)
  • Favorite system (T-dolls)
  • Search by Build Time (T-dolls)
  • Always up-to-date by using the GFL-core API
What you shouldn't expect
  • Bugs (hopefully)
